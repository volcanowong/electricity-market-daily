#### 1. 国务院办公厅《关于完善全国统一电力市场体系的实施意见》

- **发布机构 / 发布日期 / 文号**：国务院办公厅 / 2026年2月11日 / 国办发〔2026〕×号
- **核心条款**：
  1. 到2030年，基本建成全国统一电力市场体系，市场化交易电量占全社会用电量约70%；
  2. 到2035年，全面建成，市场化交易电量占比稳中有升；
  3. 推动各层次市场从"各自报价、各自交易"转向"统一报价、联合交易"；
  4. 各类型电源和除保障性用户外的电力用户全部直接参与市场；
  5. 部署5大方面19项重点任务，涵盖省间市场联合、新型经营主体接入、制度体系统一等。
- **深度解读（约350字）**：  
  本文件是2026年电力市场改革的顶层纲领，其核心逻辑是解决长期以来省间壁垒阻碍资源优化配置的核心矛盾。文件明确提出省间市场"联合报价"路径，意味着此前各省独立出清的格局将逐步被打破——跨区通道价值将随之重估，西北、东北富余新能源消纳通道的战略价值进一步提升。  
  对发电侧：强制入市压力增大，尤其是存量燃煤机组须在2030年前全部参与市场，依赖行政电价保护的空间大幅收窄；中长期合同的签约策略和对冲工具需求将快速升温。  
  对售电侧：零售市场准入门槛将更加规范统一，但同时意味着跨省购电寻价机会增多；具备跨省渠道资源的售电公司将获得差异化竞争优势。  
  对用户侧：大工业用户将被纳入直接参与市场，电价水平与现货价格联动程度加深；响应调节能力（需求侧响应、虚拟电厂）的商业价值将显著提升。  
  对储能：独立储能参与市场的制度障碍进一步清除，容量电价覆盖固定成本、现货套利覆盖变动成本的"双轨收益"路径更加清晰。  
  此文件与2025年12月发布的《电力中长期市场基本规则》构成"上下游"政策链，前者定框架，后者定规则，共同为2026年全面现货化铺路。
- **原文链接**：[人民网报道](http://politics.people.com.cn/n1/2026/0211/c1001-40664156.html) | [国家发改委解读](https://www.ndrc.gov.cn/xxgk/jd/jd/202602/t20260211_1403695.html) | [新华网全文](https://www.news.cn/politics/20260211/1472842d9a504e2796961ca387fc7b74/c.html)

#### 2. 国家发改委、国家能源局《关于完善发电侧容量电价机制的通知》

- **发布机构 / 发布日期 / 文号**：国家发展改革委、国家能源局 / 2026年1月30日 / 发改价格〔2026〕114号
- **核心条款**：
  1. 煤电、天然气发电、抽水蓄能、新型储能四类电源纳入容量电价体系；
  2. 容量电价覆盖固定成本比例从现有水平向50%以上提升；
  3. 电量收益+容量收益"双轨制"，降低单纯依赖电量市价的现金流波动风险；
  4. 容量电价标准由派出机构按地区差异核定，非全国统一；
  5. 新型储能容量电价参照抽蓄设计框架，重点考量调节能力而非装机规模。
- **深度解读（约300字）**：  
  本通知针对的核心问题是：在新能源大规模并网、现货价格信号波动剧烈的背景下，调节性电源面临"建得起、回不了本"的投资困境，导致系统灵活性不足。容量电价的本质是将"备用价值"显性化——即使不发电，也为电力系统提供了保障能力，应得到补偿。  
  对煤电：固定成本（折旧、人工、维护）有望获得更稳定覆盖，电厂面临的现金流不确定性降低，有助于防止过度退役。但另一面，倒逼机组证明自身"有效容量"，可用率和响应速度将被严格考核。  
  对新型储能：这是首次将储能纳入容量电价体系，商业模式更加完整；但"调节能力"核定标准尚在细化，短期内收益实现存在不确定性。  
  与国办2月文件形成政策链：国办文件定方向（市场化），本通知解决过渡期调节性电源收益保障问题，两者逻辑自洽。
- **原文链接**：[国家发改委官网](https://www.ndrc.gov.cn/xxgk/zcfb/tz/202601/t20260130_1403524.html) | [21经济网解读](https://www.21jingji.com/article/20260202/herald/202e60e60b1e79b041ce2d80f5ffc24f.html)

#### 3. 国家发改委、国家能源局《电力中长期市场基本规则》

- **发布机构 / 发布日期 / 文号**：国家发展改革委、国家能源局 / 2025年12月26日 / 发改能源规〔2025〕1656号 / 2026年3月1日起施行
- **核心条款**：
  1. 将分布式光伏、独立储能、虚拟电厂、智能微电网等"新型经营主体"正式纳入市场成员；
  2. 中长期市场合同期限涵盖年、季、月、周等多周期，增强合同流动性；
  3. 明确"安全约束"优先原则，现货与中长期差价结算机制规范化；
  4. 绿电交易与普通电力交易规则融合，绿证随绿电流转；
  5. 省间中长期交易通道容量分配机制规范化。
- **深度解读（约280字）**：  
  本规则解决的是市场碎片化问题——此前各省规则差异显著，新型主体入市障碍多、标准不统一，制约了全国统一市场的形成。将虚拟电厂、聚合商纳入市场成员是关键突破，意味着负荷侧响应资源可以合规参与中长期合约签订，为需求侧资源商业化打开通道。  
  绿电与绿证交易规则融合意义重大：目前绿电溢价和绿证价格存在双轨并行，本规则推动二者归一，有助于形成统一的绿色溢价信号，对光伏、风电项目在谈PPA（购售电协议）时的定价逻辑有直接影响。  
  对市场研究者而言，须重点关注"安全约束优先"条款：在新能源高比例时段，调度系统仍可基于安全原因偏离市场出清结果，这意味着价格信号并非时时有效，需区分"影子价格"与"结算价格"。

## 四、给我的研读建议（3条具体建议）

### 建议1：今天花30分钟深读——国务院办公厅《关于完善全国统一电力市场体系的实施意见》

**推荐理由**：这是2026年电力市场改革的战略顶层文件，其中"省间联合交易"、"新型经营主体入市"、"2030年70%市场化"三个核心条款将决定未来4年的市场结构走向。建议重点阅读第二、三部分（推动优化配置、健全市场功能），结合此前《电力中长期市场基本规则》对比阅读，梳理"目标-规则-机制"三层政策架构。  
**阅读切入角度**：为什么"统一报价"比"各自报价"难落地？省间利益博弈如何影响进度？这两个问题是理解后续政策推进速度的关键。  
**建议记录**：本文件的19项重点任务进度将成为未来2-3年市场监管的考核锚点，可建立追踪表格。

### 建议2：记入知识库——中电联一季度数据：太阳能装机首次超越煤电

**为什么重要**：这是电力系统结构性转变的历史性里程碑。当可再生能源装机过半时，系统"平衡逻辑"从"如何保证供给"转向"如何平衡波动"，这对现货价格信号、调峰辅助服务价值、储能投资逻辑均产生根本性影响。  
**知识库词条建议**：「2026年Q1：非化石能源装机占比突破62%，太阳能装机首超煤电，风光合计占总装机50%」→ 与电价数据、辅助服务市场容量数据关联建档。

### 建议3：追踪"算电协同"模式的可复制性与市场影响

**今日信号变化**：大唐中卫项目的商业模式（绿电直连+长协锁量+储能填谷）已初步验证，大唐发电股价强势表明资本市场认可度高。下一步关键问题是：**这个模式在其他地区是否可复制？** 制约因素是电网接入许可、碳排放置换核定、以及算力园区对绿电溢价的接受程度。  
**建议行动**：整理宁夏、内蒙古、甘肃算电协同政策支持清单，对比判断模式推广速度，并关注宁夏监管局对碳排放置换核定的具体操作细则——这将直接影响绿电直连项目的融资估值逻辑。
