#### 1. 《关于促进人工智能与能源双向赋能的行动方案》

- **发布机构 / 发布日期 / 文号**：国家发展改革委、国家能源局、工业和信息化部、国家数据局 / 2026年5月8日 / 未公开文号
- **核心条款**：
  1. 以"能源支撑AI发展、AI赋能能源转型"为双主线，部署29项重点任务
  2. 推动建立算力与电力互动机制，以电力市场价格信号引导算力设施优化跨时段用电和跨区算力调度
  3. 鼓励算力设施以多种形式参与电能量市场、辅助服务市场、需求响应，获取差价收益
  4. 新建算力设施须与可再生能源企业签订多年期绿色电力交易合同，持续提升绿电消费比例
  5. 支持绿电直连项目聚合参与电力市场，电源与负荷不同主体可分别注册
- **深度解读（约300字）**：

  此文是中国首次从国家层面以专项方案形式系统推进"AI+能源"融合，意义远超以往单部门的指导意见。其逻辑核心是：随着国内大模型训练和推理规模的指数级扩张，AI算力中心（AIDC）已成为增长最快的用电负荷类型——中国信息通信研究院预测，到2030年AIDC用电量将超过7000亿千瓦时，约占全社会用电量的5.3%。如此体量的新增负荷，若不加以市场化疏导，将对现有现货出清机制和调峰调频体系构成冲击。

  **对发电侧**：可再生能源企业将直接受益于AIDC长协需求的锁定，多年期绿电合同有助于稳定收益；风光企业还可通过绿证交易向算力设施提供"绿色溢价"。**对储能侧**：AIDC作为灵活可调负荷，与储能联合参与需求响应和辅助服务，将进一步拓宽独立储能的盈利路径。**对售电侧**：需要为大型算力客户提供绿电直购、价格风险对冲等综合服务，倒逼售电公司提升绿电资源撮合能力。**对用户侧（AIDC）**：算力设施被明确鼓励参与电力市场，意味着其需具备实时响应价格信号的负荷管理能力，EMS（能量管理系统）和AI优化调度软件需求将大幅增加。

  从政策链条看，本方案是国办发〔2026〕4号文确立统一市场框架之后，专项落地绿色算力场景的配套文件，与2025年出台的《电力中长期市场基本规则》和114号容量电价文件形成"框架—规则—场景"的完整政策体系。

- **原文链接**：https://www.nea.gov.cn/20260508/4dae97ca01d348e4871bb8654be34b3a/c.html

#### 2. 《国务院办公厅关于完善全国统一电力市场体系的实施意见》（国办发〔2026〕4号）

- **发布机构 / 发布日期 / 文号**：国务院办公厅 / 2026年2月11日 / 国办发〔2026〕4号
- **核心条款**：
  1. 到2030年基本建成全国统一电力市场体系，市场化交易电量占全社会用电量的70%左右
  2. 到2035年全面建成，除保障性用户外所有用户直接参与市场
  3. 全面建设更好发现价格、调节供需的现货市场；持续完善中长期市场；建立容量市场
  4. 完善跨省跨区电力交易制度，优化资源在全国范围内的优化配置路径
  5. 完善环境价值实现机制，构建绿电、绿证交易与碳市场的衔接机制
- **深度解读（约300字）**：

  国办4号文是继2015年9号文（新一轮电改启动）之后最高级别的电力市场顶层设计文件。它将"全国统一大市场"的战略目标首次明确落实到电力领域，并给出了具体时间节点和量化指标（70%市场化比例），为各省推进现货市场和跨省跨区交易提供了政治背书。

  **对市场参与者的含义**：对于仍在博弈是否真正推进现货的省份，4号文意味着行政指令到了，推进速度将加快。对于发电企业，中长期和现货市场的"双市场"格局将长期并存，基差风险（中长期合同价格与现货出清价之差）管理将成为核心竞争力之一。对于大型用户，直接参与市场是未来必经之路，尽早建立专业交易团队或委托专业售电公司具有先发优势。

  文件特别提及"建立容量市场"，结合此前114号文已对煤电、气电、储能给出容量电价机制，预计2026年内将有更多省份率先启动容量市场试点，进一步完善"能量+容量+辅助服务"三位一体的市场收益结构。

- **原文链接**：http://politics.people.com.cn/n1/2026/0211/c1001-40664156.html | 深度解读参见：https://zhuanlan.zhihu.com/p/2009971833731044673

#### 3. 《关于完善发电侧容量电价机制的通知》（发改价格〔2026〕114号）

- **发布机构 / 发布日期 / 文号**：国家发展改革委 / 2026年1月30日 / 发改价格〔2026〕114号
- **核心条款**：
  1. 煤电、天然气发电、抽水蓄能、**新型储能**四类电源均纳入容量电价体系（新型储能首次纳入）
  2. 新型储能容量电价以当地煤电容量电价为基础，按满功率连续放电时长折算，折算比例最高不超过1
  3. 容量电价机制与电力市场建设进展挂钩，各省因地制宜确定标准
  4. 电网侧独立新型储能电站可按规定申请容量电价，配储储能暂不纳入
  5. 分类完善机制，为抽水蓄能和新型储能的协同竞争提供明确政策框架
- **深度解读（约250字）**：

  114号文的战略意义在于完成了新型储能收益模式的"三角拼图"：**电能量套利收益 + 辅助服务收益 + 容量电价收益**，历史上长期缺失的第三角终于到位。对于电网侧独立储能项目而言，容量电价提供了相对稳定的"底薪"，降低了单纯依赖现货价差的风险，有助于推动储能项目的商业化规模化落地。

  **折算逻辑值得注意**：2小时储能（放电时长2h）若当地最高净负荷持续时长为4h，则折算比例为0.5，容量电价也只有煤电标准的一半。这意味着时长更长的储能（4h、6h）将在容量价值上更具竞争力，推动行业向长时储能转型的政策信号明确。

  自2026年1-2月数据来看，新型储能新增装机同比增长超470%，114号文落地后产业热度进一步提升。

- **原文链接**：https://www.ndrc.gov.cn/xxgk/zcfb/tz/202601/t20260130_1403524.html | 储能解读：https://www.cnesa.org/policy/detail/?column_id=6&id=7822

#### 4. 《关于开展2026年电力领域综合监管工作的通知》（国能发监管〔2026〕29号）

- **发布机构 / 发布日期 / 文号**：国家能源局 / 2026年4月9日 / 国能发监管〔2026〕29号
- **核心条款**：
  1. 2026年将对电力市场规则执行情况、现货市场运营合规情况开展专项监管
  2. 重点监管省份电力交易中心的市场准入、价格形成、结算流程的规范性

- **深度解读（约150字）**：

  本通知标志着中国电力监管正从"建规则"转向"强执法"阶段。随着现货市场常态化运行，市场操纵、不公平交易等行为的风险也随之上升。对发电企业而言，交易合规管理将成为必要能力；对新型经营主体，入市资格和行为规范将受到更严格审查。2026年监管工作的侧重点将影响各省市场规则执行的严格程度，是判断市场化进度的重要信号源。

### 2.3 重大项目与企业动态

## 四、给我的研读建议

**1. 今日精读推荐：《关于促进人工智能与能源双向赋能的行动方案》（30分钟深读）**

**推荐理由**：这份文件可能是未来2-3年内影响电力市场需求侧结构变化最重要的政策信号。它将AIDC从"普通大用户"升级为"可调节柔性负荷资源+绿电消费主体"，直接影响：①绿电/绿证交易量级；②需求响应市场规模；③售电公司的绿色能源服务产品设计；④虚拟电厂聚合AIDC资源的商业可行性。建议精读29项任务的条文并对照自身研究方向，梳理哪些任务处于政策"白区"（有任务无规则）、哪些已有规则支撑，白区恰恰是可以深入研究的空间。

**2. 值得记入知识库的数据/趋势**

- **广东4月现货均价533元/MWh，最高978元/MWh**：这是现货市场"极端事件"的一次典型案例，背后的成因（水电欠发+负荷快增+机组检修集中）、持续时长、对长协套保策略的冲击，是构建价格风险情景集（scenario set）的宝贵真实数据。
- **新型储能容量电价折算公式**：折算比例=放电时长÷当地最高净负荷持续时长，最高不超过1。这一公式可直接用于测算不同时长储能项目的容量电价收益，建议建立参数化测算模板。
- **绿证交易一季度增速21.19%**：结合AI行动方案的绿电消费要求，预计全年增速将超过40%，建议跟踪月度成交量数据变化趋势。

**3. 技能/研究方向信号变化**

- **信号：绿电直连项目市场参与规则正在快速成型**，各省正在探索多主体聚合参与绿电市场的制度设计。如果你正在研究绿电交易，可以重点关注"聚合主体注册规则"这个尚未统一的政策空白，这里有深度研究机会。
- **信号：批零倒挂和基差风险管理需求爆发**，广东案例提示中长期合同组合与现货出清的套保策略研究具有极高现实价值，金融工程+电力市场的交叉研究方向正在升温。
- **信号：AI能源岗位需要"电力市场+编程+政策"三维能力**，单一维度的研究人员竞争力正在下降，值得评估自身能力结构是否需要补强。
