#### 政策一：《关于促进人工智能与能源双向赋能的行动方案》

- **政策名称**: 关于促进人工智能与能源双向赋能的行动方案
- **发布机构 / 发布日期 / 文号**: 国家发展改革委、国家能源局、工业和信息化部、国家数据局 / 2026年5月8日
- **核心条款**:
  1. 到2027年初步构建支撑AI创新的安全绿色经济能源保障体系，算力设施与清洁能源互动能力显著增强
  2. 到2030年算力设施清洁能源供应保障能力和能源领域AI专项技术达到世界领先水平
  3. 共部署29项重点任务，涵盖算力设施能源供应保障、绿色低碳转型、算电高效经济协同等六大方向
  4. 首次将"能源赋能AI"与"AI赋能能源"置于同等战略地位，算力需求成为电力需求预测的新核心变量
  5. 推进全国首批大规模算电协同绿电直供项目，开放能源行业高价值AI应用场景
- **深度解读**:

  这是继2026年2月《完善全国统一电力市场体系实施意见》之后，本轮电力改革政策链上的又一重大节点。其战略意义在于三个层面：

  **第一，数据中心被纳入电力需求侧管理（DSM）的新型主体范畴**。算力设施具备7×24小时高负荷、可调节性弱、绿电偏好强等独特负荷特性，与工业用电存在本质区别。这要求现货市场在价格形成机制、辅助服务市场设计上做出新的适应——算力负荷的刚性特征可能进一步抬高广东、上海等算力聚集地的现货市场峰段电价。

  **第二，"双向赋能"的提法在政策语言层面是一次重大突破**。过去AI与电力的关系基本是单向的"AI用电"，此次政策明确AI将在需求预测、调度优化、交易决策、储能管理等方向深度落地，电力市场建模、量化分析等岗位需求将随之快速扩张，"电力+AI"复合型人才将出现显著溢价。

  **第三，绿电直连（Power Purchase Agreement, PPA）将因算力企业的绿电需求而实质性提速**。当前全国已审批的99个绿电直连项目中，民营企业占比过半，对应装机超1400万千瓦；算力企业同样有强烈绿电采购动机，这将推动电价相对较低的西北、西南省份（新疆、青海、云南）成为绿电供应新中心。

  对市场参与者：**发电侧**（特别是风光）受益于算力绿电需求拉动；**售电公司**需要新增面向数据中心的绿电采购套餐产品；**储能**将在算力园区"电力保障+调峰"双功能中找到新市场；**电网侧**面临数据中心负荷聚集性增长的消纳压力，传输通道规划亟需考虑算力布局。

- **原文链接**: https://www.nea.gov.cn/20260508/4dae97ca01d348e4871bb8654be34b3a/c.html

#### 政策二：《关于完善全国统一电力市场体系的实施意见》（重要背景政策）

- **政策名称**: 关于完善全国统一电力市场体系的实施意见
- **发布机构 / 发布日期 / 文号**: 国务院办公厅 / 2026年2月11日 / 国办发〔2026〕4号
- **核心条款**:
  1. 2030年基本建成全国统一电力市场体系，2035年全面建成
  2. 首次提出各层级市场由"自定价自交易"向"统一定价联合交易"渐进转变
  3. 探索相邻省份市场自愿协调衔接或合并的可行路径（省间整合先例）
  4. 明确现货、中长期、辅助服务、容量市场四类市场的互联衔接机制
  5. 推动可靠容量市场（Capacity Market）建设，按顶峰能力对机组可靠容量给予补偿
- **深度解读**:

  这是迄今为止电力市场顶层设计中**级别最高（国务院办公厅）、覆盖范围最广**的一份文件。与2015年"9号文"、2021年双碳背景下的电改加速相比，国办4号文的核心突破在于**将"省间价格统一"从长期目标变为阶段性行动路径**——要求相邻省市探索联合出清、统一定价，这意味着当前省级现货市场的孤岛化运行模式将在5年内面临根本性变革。

  对当前依赖省内分时价差套利的储能运营商而言，省间套利空间将随之打开，但省内峰谷差套利空间可能因市场范围扩大而受到压缩。对电力交易员来说，未来的市场博弈从单省扩展到跨省区域，对全网感知能力和跨市场策略能力的要求将大幅提升。

  可靠容量市场的建立同样值得深度跟踪——这是中国电力市场迄今缺失的重要一环。英国、美国PJM市场的容量市场实践表明，该机制将深刻改变调峰电源、储能的盈利结构和投资回报预期。

- **原文链接**: http://politics.people.com.cn/n1/2026/0211/c1001-40664156.html

#### 政策三：《关于完善发电侧容量电价机制的通知》（重要背景政策）

- **政策名称**: 关于完善发电侧容量电价机制的通知
- **发布机构 / 发布日期 / 文号**: 国家发展改革委、国家能源局 / 2026年1月30日 / 发改价格〔2026〕114号
- **核心条款**:
  1. 煤电机组固定成本通过容量电价回收的比例提升至不低于50%（此前约30%）
  2. 电网侧独立新型储能电站**首次纳入国家层面容量电价机制**，以煤电容量电价标准为基础按顶峰能力折算
  3. 气电容量电价机制同步建立，解决气电机组长期缺乏稳定收益的问题
  4. 现货市场连续运行后，有序建立发电侧可靠容量补偿机制
  5. 发电侧整体向"电量收益+容量收益"双轨制收益结构全面转型
- **深度解读**:

  这份通知对储能行业的意义是根本性的：**它首次将电网侧独立新型储能的商业模式从"依赖辅助服务+现货套利的不确定性收益"转向"容量电价保底+市场收益叠加"的稳定结构**。过去储能项目的财务模型难以获得银行贷款，根本原因在于收益预期过于依赖政策变化和现货价格波动；有了容量电价兜底，储能投资的内部收益率（IRR）预测区间将收窄，社会资本进入速度有望加快。

  但需注意两个结构性约束：**一是折算顶峰能力的容量电价设计实质上激励更长时长储能**（4小时以上优于2小时），对市场上大量在建的2小时短时储能形成竞争压力；**二是煤电获得50%固定成本回收后，其参与现货市场低谷时段的压价动机将减弱**，低谷电价可能进一步下探——这对需要低价充电的储能套利模型是一个重要利好。

- **原文链接**: https://www.ndrc.gov.cn/xxgk/zcfb/tz/202601/t20260130_1403524.html

#### 政策四：《电力中长期市场基本规则》（执行中）

- **政策名称**: 电力中长期市场基本规则
- **发布机构 / 发布日期 / 文号**: 国家发改委、国家能源局 / 2025年12月26日发布 / 发改能源规〔2025〕1656号 / 2026年3月1日起施行
- **核心条款**:
  1. 取消对直接参与市场交易经营主体的固定分时电价时段限制（市场化用户不再适用目录分时电价）
  2. 建立中长期与现货市场价格联动机制
  3. 规范中长期合约签约比例与形式（实物合约与差价合约并轨）
  4. 有效期5年，并要求各地3月1日前完成实施细则备案
  5. 全国已有9个地区落地取消固定分时电价政策
- **深度解读**:

  取消固定分时电价对市场化用户是本轮规则中影响最深远的条款之一。原有分时电价是人为划定峰谷时段、设置固定价格比例，在现货市场充分发育后，这种机制本质上是对价格信号的压制。**取消固定分时电价后，市场化用户必须真正面对现货市场的实时价格波动**，这要求工业用户配备更专业的用电管理能力，也推动虚拟电厂（VPP）和需求响应聚合商的价值快速显现。

- **原文链接**: https://www.ndrc.gov.cn/xxgk/zcfb/ghxwj/202512/t20251226_1402666.html

## 二、市场动态

### 2.1 现货与中长期价格信号

#### 广东市场（本周最值得关注）

| 指标 | 数值 | 备注 |
|------|------|------|
| 2026年市场参考价 | 0.463元/kWh | 年度基准 |
| 4月实时均价峰值 | ~0.536元/kWh（南方区域） | 历史次高，仅次于2022年 |
| 4月阶段性最高 | 逼近1元/度 | 多次出现，历史次高纪录 |
| 年度交易价格上下限 | 0.372—0.554元/kWh | 2026年设定 |

**分析**：广东4月高电价的成因是三重因素叠加——华南用电需求超预期增长（+AI算力中心）、清洁能源出力阶段性波动、设备集中检修削减可用容量。进入5月，华南用电旺季正式开启，气温持续攀升将进一步推高空调负荷，叠加算力中心用电持续攀升，广东现货均价有望在本月维持0.45—0.55元/kWh的高位区间，峰谷价差扩大趋势明显。

#### 山东市场

- 现货市场常态化运行稳定；春末负荷回升，光伏出力充裕
- 低谷时段（0:00—6:00）光伏不发电，风电出力偏高，低谷价格可能进一步走低
- 预计5月中下旬随高温来临，日间峰段价格进一步攀升，峰谷价差有望扩大至200元/MWh以上

#### 山西市场

- 现货市场正式运行，煤电占比高，火电机组受容量电价改革影响，低谷报价策略出现调整信号
- 需关注新能源消纳压力变化

#### 蒙西/甘肃市场

- 五月北方晴好天气下，光伏发电量屡创阶段性新高
- 新能源出力增加叠加负荷有限，负电价出现频率有上升趋势
- 跨省输送通道约束仍是消纳主要瓶颈；《完善全国统一电力市场体系实施意见》提出跨省优化配置方向，对蒙西/甘肃长期消纳有实质性利好

#### 浙江/江苏市场

- 两省电力需求稳定增长，现货市场运行在合理区间
- 江苏已完成电力现货市场连续结算试运行切换，市场规则执行趋于成熟
- 安徽现货新规实施将推动华东区域市场联动性增强

#### 综合价格信号解读

**高频技能关键词（Top 5）**：
1. **Python / 数据分析**：分析类岗位全系必备，含电价预测、市场分析、量化交易

## 四、给我的研读建议（3条具体建议）

**1. 今天值得花30分钟深读的政策**

**推荐深读**：《关于促进人工智能与能源双向赋能的行动方案》（2026年5月8日）。

**理由**：这是本周发布的最新国家级政策，且在电力市场研究领域尚属新命题，市场解读处于早期阶段。29项任务中有多项与电力市场直接相关——建议重点精读"第三部分：推进算电高效经济协同"，尤其关注以下具体条款：①数据中心作为需求侧响应新型主体参与现货市场的制度安排；②绿电直连项目的定价与结算规则设计；③算力聚集区输电通道规划的协调机制。这三个条款是理解未来5年广东、长三角电力市场结构性变化的关键切入口。

原文链接：https://www.nea.gov.cn/20260508/4dae97ca01d348e4871bb8654be34b3a/c.html

---

**2. 值得记入电力市场知识库的数据/趋势**

**推荐记录**：广东现货市场4月实时均价阶段性逼近1元/度（南方区域实时均价峰值约0.536元/kWh），与年度市场参考价0.463元/kWh相比，溢价率约16%；与2026年度年度上限0.554元/kWh仅差约3%，接近规则触发区间。

建议知识库记录维度：
- **触发因素三要素**：气温超预期（华南早热）、集中检修削减可用容量、算力中心新增负荷
- **历史对比锚点**：历史最高为2022年1.07元/度，当前为次高水平
- **价格与政策关联性**：高价格背景下广东容量电价、需求响应政策实施效果评估
- **后续观测指标**：5月旺季是否突破历史高点？储能调峰能否平抑峰值电价？

这是分析广东未来容量电价机制效果、测算储能套利空间的重要基准数据。

---

**3. 今日有信号变化的技能/研究方向**

**信号升温方向**："算电协同电力市场建模"。

《双向赋能行动方案》首次将AI算力中心的负荷特性（连续高负荷、绿电偏好强、可响应能力弱）纳入能源规划框架，这意味着**传统电力需求预测模型必须加入"算力增长曲线"这一新变量**。当前市场上几乎没有研究者将算力规模与电力市场价格形成机制进行系统性连接。

**建议今日开始积累的知识点**：
1. 数据中心用电特征库（负荷曲线平坦度、功率因数、可中断能力）
2. 全国算力中心地理分布与当地电力市场（电价水平、绿电资源）的交叉匹配
3. 典型算力园区绿电直连项目的商务结构（PPA电价、期限、绿证归属）
4. AI用电增长弹性测算方法（从GPU算力规模映射到年用电量）

掌握以上方向，将成为未来2—3年电力市场分析师极具差异化竞争力的核心能力，也是当前招聘方（AI+能源公司、头部售电公司）愿意为之支付溢价的稀缺技能。
